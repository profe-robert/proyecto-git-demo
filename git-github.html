Conecta tu repositorio local con GitHub
git remote add origin [URL_REPO]

Agregar Archivos

git add .
arrow_downward

Confirmar Cambios
Guarda los cambios en el repositorio local
git commit -m "mensaje"
arrow_downward

Obtener Cambios
Descarga cambios del repositorio remoto
git pull origin main
arrow_downward

Envía tus commits al repositorio remoto
git push origin main
